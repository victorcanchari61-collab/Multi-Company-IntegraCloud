# HRM — Human Resource Management

## Descripción General
Sistema que administra todos los procesos relacionados con el capital humano, desde la contratación hasta la administración de nómina y desarrollo de talento.

## Módulos y Submódulos

### 1. Reclutamiento
| Submódulo | Descripción | Flujo de trabajo | Requisitos clave |
|-----------|-------------|------------------|------------------|
| Publicación de vacantes | Difusión de ofertas laborales | Solicitud → Aprobación → Descripción → Publicación → Canales | Bolsas de trabajo, redes sociales, web corporativa, referidos |
| Gestión de candidatos (ATS) | Seguimiento de postulaciones | Recepción → Filtro → Clasificación → Pipeline → Base de datos | Curriculum parsing, scoring, filtros automáticos, comunicación |
| Entrevistas y evaluaciones | Proceso de selección | Preselección → Entrevista → Prueba → Evaluación → Feedback | Agenda, formato de evaluación, panel de entrevistadores, rubricas |
| Onboarding | Integración del nuevo empleado | Documentación → Inducción → Equipo → Recursos → Primeros días | Checklist de onboarding, asignación de mentor, capacitación inicial |

### 2. Nómina
| Submódulo | Descripción | Flujo de trabajo | Requisitos clave |
|-----------|-------------|------------------|------------------|
| Cálculo de nómina | Procesamiento de sueldos y salarios | Novedades → Asistencia → Cálculo → Validación → Pago | Sueldo base, horas extra, comisiones, bonos, deducciones |
| Beneficios y deducciones | Gestión de prestaciones | Configuración → Cálculo → Aplicación → Reporte | Seguro social, seguro médico, fondo de ahorro, vales |
| Horas extra | Control y pago de tiempo extra | Registro → Autorización → Cálculo → Pago | Tarifas, topes, tipos de hora extra, aprobación |
| Reportes legales | Cumplimiento de obligaciones fiscales/laborales | Datos → Reglas → Cálculo → Reporte → Envío | CFDI, timbrado, declaraciones, recibos de nómina |
| Integración bancaria | Dispersión de pagos | Nómina → Archivo bancario → Dispersión → Confirmación | SPEI, layout bancario, validación de cuentas |

### 3. Desempeño
| Submódulo | Descripción | Flujo de trabajo | Requisitos clave |
|-----------|-------------|------------------|------------------|
| Evaluación 360° | Retroalimentación de múltiples fuentes | Diseño → Evaluadores → Evaluación → Consolidación → Feedback | Competencias, objetivos, autoevaluación, peers, subordinados |
| Objetivos y OKRs | Gestión por objetivos | Definición → Alineación → Seguimiento → Evaluación → Resultados | OKRs en cascada, check-ins, KR numéricos, dueños |
| Plan de desarrollo | Crecimiento profesional | Diagnóstico → Metas → Acciones → Recursos → Seguimiento | Planes individuales, cursos, mentoría, hitos |
| Retroalimentación continua | Feedback en tiempo real | Evento → Comentario → Registro → Historial → Anual | Reconocimientos, áreas de mejora, momentos destacados |

### 4. Capacitación
| Submódulo | Descripción | Flujo de trabajo | Requisitos clave |
|-----------|-------------|------------------|------------------|
| Planes de formación | Programas de capacitación | Detección → Plan → Programación → Inscripción → Ejecución | Mapa de habilidades, brechas, presupuesto, calendarización |
| LMS integrado | Plataforma de aprendizaje | Cursos → Asignación → Progreso → Evaluación → Certificación | Contenido SCORM, exámenes, foros, gamificación |
| Certificaciones | Gestión de certificaciones | Requisito → Inscripción → Curso → Examen → Certificado | Vigencia, renovación, créditos, acreditación |
| Historial de capacitación | Registro de formación recibida | Curso → Participante → Resultado → Documento → Historial | Transcript, horas acumuladas, inversión, efectividad |

### 5. Asistencia
| Submódulo | Descripción | Flujo de trabajo | Requisitos clave |
|-----------|-------------|------------------|------------------|
| Control de asistencia | Registro de entrada/salida | Marcación → Validación → Registro → Reporte → Novedades | Reloj checador, biometrico, móvil, web, tolerancia |
| Gestión de turnos | Administración de horarios | Patrón → Asignación → Rotación → Cambio → Cobertura | Turnos fijos, rotativos, nocturnos, jornada acumulativa |
| Vacaciones y permisos | Solicitud y aprobación de ausencias | Solicitud → Saldo → Aprobación → Registro → Pago | Políticas por tipo, saldo disponible, antigüedad |
| Ausentismo | Monitoreo de inasistencias | Registro → Causa → Reporte → Análisis → Intervención | Índice de ausentismo, tendencias, justificaciones |
