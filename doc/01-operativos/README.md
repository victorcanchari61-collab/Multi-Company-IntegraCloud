# Sistemas Operativos

Los sistemas operativos ejecutan los procesos core del negocio. Son el corazón transaccional de la empresa y gestionan las operaciones diarias.

## Sistemas incluidos

| Sistema | Nombre completo | Descripción |
|---------|-----------------|-------------|
| [ERP](ERP-PlanificacionRecursos.md) | Enterprise Resource Planning | Integra todos los procesos core del negocio en una sola plataforma |
| [CRM](CRM-GestionRelacionesCliente.md) | Customer Relationship Management | Gestiona las relaciones con clientes a lo largo del ciclo de vida |
| [WMS](WMS-GestionAlmacenes.md) | Warehouse Management System | Controla y optimiza las operaciones dentro del almacén |
| [SCM](SCM-CadenaSuministro.md) | Supply Chain Management | Coordina el flujo de bienes, información y finanzas en la cadena de suministro |
| [MRP](MRP-PlanificacionMateriales.md) | Material Requirements Planning | Calcula los materiales necesarios para la producción |
| [TMS](TMS-GestionTransporte.md) | Transportation Management System | Gestiona la planificación, ejecución y optimización del transporte |
| [HRM](HRM-GestionHumana.md) | Human Resource Management | Administra todos los procesos relacionados con el capital humano |
| [POS](POS-PuntoVenta.md) | Point of Sale | Gestiona las transacciones de venta en el punto de contacto con el cliente |
| [PLM](PLM-CicloVidaProducto.md) | Product Lifecycle Management | Gestiona el ciclo de vida completo del producto |
| [QMS](QMS-GestionCalidad.md) | Quality Management System | Asegura que productos y procesos cumplan estándares de calidad |
| [EAM](EAM-GestionActivos.md) | Enterprise Asset Management | Gestiona el ciclo de vida de los activos físicos |
