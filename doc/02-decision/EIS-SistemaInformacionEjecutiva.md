# EIS — Executive Information System

## Descripción General
Sistema que provee información estratégica y resumida a la alta dirección, facilitando el monitoreo del desempeño organizacional y la identificación de tendencias clave.

## Módulos y Submódulos

### 1. Monitoreo estratégico
| Submódulo | Descripción | Flujo de trabajo | Requisitos clave |
|-----------|-------------|------------------|------------------|
| Balanced Scorecard (BSC) | Tablero de control integral | Perspectivas → Objetivos → KPIs → Metas → Iniciativas → Seguimiento | Financiera, cliente, procesos, aprendizaje; relación causa-efecto |
| KPIs estratégicos | Indicadores clave de la organización | Definición → Línea base → Meta → Medición → Reporte | KPIs financieros, operativos, comerciales; alineación con estrategia |
| Semáforos de desempeño | Alertas visuales de estado | KPI → Umbral → Verde/Amarillo/Rojo → Visualización → Acción | Reglas de semáforo, tendencias, comparativo con meta |
| Tendencias ejecutivas | Visualización de evolución de métricas | Histórico → Serie → Tendencia → Proyección → Alerta | Líneas de tendencia, media móvil, estacionalidad, cambio de tendencia |

### 2. Análisis financiero
| Submódulo | Descripción | Flujo de trabajo | Requisitos clave |
|-----------|-------------|------------------|------------------|
| P&L ejecutivo | Estado de resultados resumido | Ingresos → Costos → Margen → Gastos → Utilidad → EBITDA | Comparativo vs presupuesto/año anterior, márgenes porcentuales |
| Flujo de caja | Monitoreo de liquidez | Ingresos → Egresos → Operación → Inversión → Financiamiento → Saldo | Proyección a 13 semanas, saldo mínimo alerta, fuentes/ usos |
| Comparativo vs presupuesto | Desviación entre real y presupuestado | Real → Presupuesto → Desviación % → Análisis → Acción | Variación favorable/desfavorable, explicación, forecast |
| Análisis de rentabilidad | Rentabilidad por unidad de negocio | Ingresos → Costos directos → Margen → Costos indirectos → Utilidad | Por producto, cliente, región, canal; rentabilidad neta |

### 3. Alertas ejecutivas
| Submódulo | Descripción | Flujo de trabajo | Requisitos clave |
|-----------|-------------|------------------|------------------|
| Umbrales de alerta | Configuración de límites críticos | KPI → Límite → Condición → Prioridad → Alerta | Alertas por exceso/defecto, porcentaje, valor absoluto, tendencia |
| Notificaciones automáticas | Comunicación proactiva de eventos | Evento → Regla → Destinatario → Canal → Mensaje | Email, SMS, push, dashboard; escalamiento por gravedad |
| Resúmenes diarios/semanales | Reportes periódicos ejecutivos | Período → Datos → Formato → Distribución → Lectura | Formato compacto, métricas principales, cambios relevantes |
| Excepciones críticas | Identificación de anomalías significativas | Normal → Desviación → Excepción → Investigación → Acción | Regla de excepción, materialidad, impacto, prioridad de atención |
