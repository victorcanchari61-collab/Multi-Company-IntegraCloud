# Sistemas de Decisión

Los sistemas de decisión analizan datos para apoyar la toma de decisiones estratégicas, tácticas y operativas en la organización.

## Sistemas incluidos

| Sistema | Nombre completo | Descripción |
|---------|-----------------|-------------|
| [BI](BI-InteligenciaNegocios.md) | Business Intelligence | Transforma datos en información accionable para la toma de decisiones |
| [DSS](DSS-SoporteDecisiones.md) | Decision Support System | Apoya decisiones complejas combinando modelos analíticos y datos |
| [EIS](EIS-SistemaInformacionEjecutiva.md) | Executive Information System | Provee información estratégica y resumida a la alta dirección |
