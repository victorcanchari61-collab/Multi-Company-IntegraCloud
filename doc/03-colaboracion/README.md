# Sistemas de Colaboración

Los sistemas de colaboración conectan personas, facilitan la comunicación y el trabajo en equipo, y automatizan tareas administrativas.

## Sistemas incluidos

| Sistema | Nombre completo | Descripción |
|---------|-----------------|-------------|
| [KMS](KMS-GestionConocimiento.md) | Knowledge Management System | Captura, organiza y distribuye el conocimiento organizacional |
| [ECS](ECS-ColaboracionEmpresarial.md) | Enterprise Collaboration System | Facilita la comunicación y trabajo en equipo |
| [OAS](OAS-AutomatizacionOficina.md) | Office Automation System | Automatiza tareas administrativas y de oficina |
| [PRM](PRM-GestionSocios.md) | Partner Relationship Management | Gestiona las relaciones con socios y canales de venta |
